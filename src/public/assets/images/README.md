# Logo Image Folder

Please upload your logo file here as **logo.png**

## File Requirements:
- File name: `logo.png`
- Recommended size: 200x60px or similar aspect ratio
- Format: PNG (with transparent background recommended)

## Location:
The logo will appear in:
1. Top navbar
2. Chat widget avatar

After uploading the logo, run:
```bash
npm run build
```

Or if the dev server is running, it will auto-reload with the new logo.
